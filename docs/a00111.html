Detailed Description

Represents a PCL error context, with optional stack trace.

This can be thought of as Unix errno on steroids.

◆ frozen

int frozen

‪Freezes context, making it immutable.

This value can only be 0 or 1. It should be noted that a frozen context object cannot be cleared or freed. Typically, freezing an object is only useful when used with the PCL error API, For example: freezing the per-thread error context before performing cleanup after an error, where the cleanup could overwrite the "real" error: ie. freeze=1; cleanup_code; freeze=0;. Internally, all printf-style context functions freeze the object before performing output and thaw before returning.
