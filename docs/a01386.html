◆ pcl_json_obj()

PCL_PUBLIC pcl_json_t* pcl_json_obj ( void  )

#include <pcl/json.h>

‪Create an object.

It is safe to use the hash table module for managing a JSON object. Internally, a json object is backed by a pcl_htable_t. For a given pcl_json_t, ensure the type member is 'o' and then use json->object with the htable functions. However, do not use pcl_htable_free.

Returns
‪pointer to a json object of type object
