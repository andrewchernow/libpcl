35 /** @defgroup string String Management
36  * A set of string utility functions. The PCL string library includes many standard C string
37  * functions for the sole purpose of providing buffer safe versions. There are other functions
38  * that have no counterpart in the standard C library, but are a common need.
39  *
40  * Almost all PCL string functions come in two flavors: UTF-8 (char)
41  * and UTF-16 (wchar_t). These two version exists to support PCL's concept of a ::pchar_t: a
42  * \c char on Unixes and \c wchar_t on Windows. For example, the below shows some basic mappings:
43  * @code
44  * pcl_pcscpy => Unix pcl_strcpy, Windows pcl_wcscpy
45  * pcl_pcsdup => Unix pcl_strdup, Windows pcl_wcsdup
46  * pcl_pcscatf => Unix pcl_strcatf, Windows pcl_wcscatf
47  * @endcode
48  * It is actually uncommon to use \c str or \c wcs versions directly in a cross-platform
49  * application. It is much more common to use the \c pcs versions.
50  * @{
51  */
