◆ pcl_strtoi()

PCL_PUBLIC int pcl_strtoi ( const char *  nptr,
char **  end,
int  base,
int *  r 
)

#include <pcl/strint.h>

‪Convert a character string to an integer.

The integer conversion functions do range checks in addition to the initial conversion. If the converted value is out of range, -1 is returned and PCL_ERANGE is set.

Parameters
nptr‪pointer to a character string
end‪a pointer that if provided, will point to the first invalid character
base‪integer base between 2 and 36 inclusive or special value of 0. When 0, if nptr begins with '0x', base 16 is used. If nptr begins with '0', octal is used. Otherwise, base 10 is used.
r‪pointer to the result of the conversion
Returns
‪0 on success and -1 on error.

Referenced by pcl_json_compile().
