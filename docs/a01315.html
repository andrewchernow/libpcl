◆ pcl_json_arr()

PCL_EXPORT pcl_json_t* pcl_json_arr ( void  )

#include <pcl/json.h>

‪Create an array object.

It is safe to use the array module for managing a JSON array. Internally, a json array is backed by a pcl_array_t. For a given pcl_json_t, ensure the type member is 'a' and then use json->array with the array functions. However, do not use pcl_array_free.

Returns
‪pointer to a json object of type array
