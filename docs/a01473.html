◆ cleanup

pcl_cleanup_t cleanup

‪Element cleanup callback.

Warning
‪DO NOT set this to pcl_cleanup_ptr. That function frees the element argument but vector elements are owned by the vector. A vector cleanup function should free allocated memory contained within the element: like a struct member.

Example

static void mycleanup(void *elem)
{
myobj_t *o = elem; // NEVER free 'elem', owned by vector
pcl_free(o->str);
}
