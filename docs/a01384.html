◆ pcl_json_decode()

PCL_PUBLIC pcl_json_t* pcl_json_decode ( const char *  json,
size_t  len,
const char **  end 
)

#include <pcl/json.h>

‪Decode a JSON string.

This parses a JSON string that represents a single JSON value. A JSON value can be an object {}, array [], string "", number -102.84, boolean true or false or a null null. Parsing stops when the parsing of the JSON value at json is complete, regardless of the len argument. However, if a complete JSON value is not found before reaching json + len, an error occurs. The json string can contain bogus data after the value being parsed, since the parser stops at the end of the value. For example:

{
"name": "Henry Ford"
}#invalid json here

The above is completely valid, since a valid JSON object exists and ends before any invalid JSON text is encountered. When the end argument is supplied, it would be pointing at the '#' in the above example. This is very useful if a string contains multiple JSON values. pcl_json_decode can be called multiple times by passing the value of end as json, allowing one to iterate through a series of json values.

Parameters
json‪pointer to a json string. This must be NUL terminated if len is 0.
len‪number of bytes within json argument. If 0, json must be NUL terminated.
end‪pointer to the first character not parsed. This can be NULL.
Returns
‪pointer to a json value or NULL on error
