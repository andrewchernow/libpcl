◆ pcl_cipher_reset()

PCL_EXPORT int pcl_cipher_reset ( pcl_cipher_t *  ciph,
const void *  iv,
const void *  aad,
uint32_t  aad_len 
)

#include <pcl/crypto.h>

‪Reset a cipher so it can be reused.

This should be called after calling pcl_cipher_final.

This requires providing a new IV since using the same IV across cycles is a very bad idea. The key cannot be changed. If the key needs to be changed, free this cipher and create a new one.

The goal of reset is to avoid tearing everything down and creating a whole new context when it is desired to reuse the same KEY: set of files, network packets, etc. Instead, it can be reset with a new IV and optional AAD data. After this call, use pcl_cipher_update and pcl_cipher_final as usual.

Parameters
ciph‪pointer to a cipher object
iv
aad‪additional authenticated data. This is only used by ciphers that support authenticated encryption with associated data (AEAD). This can be NULL in either case.
aad_len‪number of aad bytes. If zero, aad parameter is ignored.
Returns
‪0 for success and -1 on error.
