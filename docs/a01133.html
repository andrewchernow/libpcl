◆ val

int val

‪A value, like a character, that can be used to identify the long option.

This is simpler to work with (like a switch) then comparing long option names. It is common to set this value to the cooresponding short option.

Warning
‪This value should not be -1 or 0, as these are pcl_getopt return values used to, respectively, indicate an error or completion.
