Detailed Description

A JSON value object.

This represents any valid JSON value: boolean, number (integer or real), string, null, array or object.

◆ type

char type

‪JSON value type.

This can be: 0 for null, 'r' for real, 'i' for integer, 'b' for bool, 's' for string, 'a' for array, 'o' for object. Both real and integer are actually a JSON number but are separated for simplicity. There are macros for detecting value type that all begin with pcl_json_is.
