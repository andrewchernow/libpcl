77 /** Decode a JSON string. This parses a JSON string that represents a single JSON value. A JSON
78  * value can be an object \c {}, array \c [], string \c "", number \c -102.84, boolean \c true or
79  * \c false or a null \c null. Parsing stops when the parsing of the JSON value at \a json
80  * is complete, regardless of the \a len argument. However, if a complete JSON value is not found
81  * before reaching \a json \c + \a len, an error occurs. The \a json string can contain bogus data
82  * after the value being parsed, since the parser stops at the end of the value. For example:
83  * @code
84  * {
85  * "name": "Henry Ford"
86  * }#invalid json here
87  * @endcode
88  * The above is completely valid, since a valid JSON object exists and ends before any invalid
89  * JSON text is encountered. When the \a end argument is supplied, it would be pointing at
90  * the '#' in the above example. This is very useful if a string contains multiple JSON values.
91  * pcl_json_decode can be called a multiple times passing the value of \a end as \a json,
92  * allowing one to iterate through a series of json values.
93  *
94  * @param json pointer to a json string. This must be NUL terminated if \a len is 0.
95  * @param len number of bytes within \a json argument. If 0, \a json must be NUL terminated.
96  * @param end pointer to the first character not parsed. This can be \c NULL.
97  * @return pointer to a json value or \c NULL on error
98  */
99 PCL_EXPORT pcl_json_t *pcl_json_decode(const char *json, size_t len, const char **end);
