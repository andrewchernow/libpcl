Detailed Description

Represents a PCL error context, with optional stack trace.

This can be thought of as Unix errno on steroids.

◆ frozen

bool frozen

‪Freezes context, making it immutable.

It should be noted that a frozen context object cannot be cleared or freed. The only exception is the per-thread context, which can and will be freed if the thread exits.

Typically, freezing an object is only useful when used with the PCL error API. For example: freezing the per-thread context before performing cleanup after an error, where the cleanup could overwrite the "real" error: ie. freeze=true; cleanup_code; freeze=false;.

Internally, all printf-style context functions freeze the object before performing output and thaw before returning.
